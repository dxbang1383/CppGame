# VS Code

Clone dự án về máy, sau đó mở thư mục gốc của dự án bằng VS Code.
Cài các extension được VS Code gợi ý, hoặc cài thủ công:
+ CMake Tools
+ C/C++ (đều của Microsoft)
+ 
Nhấn Ctrl + Shift + P → chọn CMake: Select Configure Preset → chọn preset x64-debug.
Nếu VS Code hỏi Build Preset, hãy chọn [Default].

Nhấn F7 để biên dịch (Build).
Nhấn Ctrl + F5 để chạy chương trình (không debug), hoặc F5 để chạy kèm trình gỡ lỗi (Debug).
